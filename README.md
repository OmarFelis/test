Funcionalidades Agregadas
1. Filtros Funcionales
Botones activos que cambian de color al seleccionar

Filtrado dinámico por: Todos, Nuevo, Usado, Más Vendido

Renderizado en tiempo real del catálogo

2. Modal de Detalles
Botón "Ver detalles" en cada vehículo

Modal responsive con información completa

Especificaciones técnicas (motor, combustible, etc.)

Cerrar con X o clic fuera del modal

3. Carrito de Cotización
Botón "Cotizar" en el modal

Contador visual en el header (rojo)

LocalStorage para persistir datos

Notificaciones de confirmación

4. Búsqueda Básica
Botón de búsqueda funcional

Filtrado por modelo/tipo

Scroll automático a resultados

5. Datos Estructurados
Array de vehículos con toda la información

Estados: nuevo, usado, vendido

Especificaciones completas

Cómo Funciona
Filtros: Clic en botones filtra vehículos instantáneamente

Modal: Clic en "Ver detalles" abre ventana con info completa

Cotización: Agrega vehículos a lista persistente

Búsqueda: Busca por modelo desde el formulario
